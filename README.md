
# Customer Lifetime Value Prediction Using Machine Learning: A Comparative Analysis
# Overview
This study addresses the prediction of Customer Lifetime Value (CLV) through the application of machine learning techniques to transactional e-commerce data. The central aim is to examine how contemporary machine learning approaches compare against conventional customer valuation methods in forecasting future revenue generated by individual customers.
# Research Objectives
This project pursues four primary objectives: to estimate customer lifetime value from historical transactional data; to benchmark traditional RFM-based valuation against supervised machine learning models; to determine the behavioral variables that most significantly drive customer value; and to assess model accuracy through established regression evaluation metrics.
# Dataset
The analysis draws on the UCI Online Retail Dataset, which comprises transactional records from a UK-based e-commerce retailer. The core variables include customer identification, invoice date, quantity purchased, and unit price — collectively enabling the derivation of customer-level behavioral patterns.

# Methodology
Data Preprocessing. Initial data preparation involved the removal of records with missing customer identifiers and the exclusion of negative transactions attributable to product returns. A revenue variable was subsequently computed for use in downstream analysis.
Feature Engineering. Customer behavior was operationalized through a set of constructed features encompassing recency (days elapsed since the most recent purchase), transaction frequency, cumulative monetary expenditure, average order value, customer tenure, inter-purchase interval statistics, and purchase momentum — reflecting directional shifts in buying behavior over time.
Temporal Splitting. A temporal partitioning strategy was adopted whereby an observation window was used for feature construction, while a subsequent prediction window served as the basis for the target variable — representing future CLV.
# Models Evaluated
Four modeling approaches were employed: an RFM-based baseline representing the traditional valuation paradigm, alongside three supervised regression models — Linear Regression, Random Forest, and XGBoost.
Evaluation Metrics
Model performance was assessed using Mean Absolute Error (MAE), Root Mean Squared Error (RMSE), and the coefficient of determination (R²).
# Results
Across all metrics, machine learning models demonstrated superior predictive performance relative to the RFM baseline. Random Forest yielded the most favorable results, achieving the lowest RMSE and the highest R² value, suggesting a modest but meaningful capacity to explain variance in future customer revenue.
# Discussion and Key Insights
The findings indicate that behavioral features — particularly purchase frequency, recency, and spending magnitude — constitute significant predictors of CLV. Nevertheless, prediction accuracy remained moderate across all models, reflecting the inherent variability and stochastic nature of individual consumer behavior. While RFM segmentation retains utility as a descriptive and diagnostic tool, its predictive limitations become apparent in regression settings where precise revenue estimation is required.
# Visualizations
The project incorporates several analytical visualizations, including RFM distribution plots, a feature correlation heatmap, a cross-model performance comparison, feature importance rankings, and a predicted versus actual CLV scatter plot.
# Conclusions
Several conclusions emerge from this work. First, customer purchasing behavior is difficult to predict with high precision, and moderate R² values should be interpreted within this context. Second, the quality and depth of feature engineering significantly influence model outcomes. Third, ensemble methods such as Random Forest demonstrate particular suitability for structured tabular data. Finally, while RFM remains a widely used segmentation framework, its adoption as a standalone predictive mechanism is inadvisable when quantitative forecasting accuracy is the primary concern.
